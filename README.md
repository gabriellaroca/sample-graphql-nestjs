## Description

GraphQL Sample Application with NestJS

## Installation

```bash
$ pnpm install
```

## Environment variables

We need configure the environment variables to run the application.

```bash
# .env
SPRING_APP_BASE_URL=http://localhost:8080 # base url of the back-end application
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
